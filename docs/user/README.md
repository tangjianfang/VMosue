## About VMosue

VMosue lets you control the mouse cursor on a Windows 10/11 PC with hand
gestures captured by your webcam. There is nothing to wear: no gloves,
no controllers, no special hardware.

Right hand moves the cursor and clicks. Left hand scrolls and pauses.
Either hand can trigger an emergency stop.
